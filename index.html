Exercise 1
- Cuando la cadena está vacía, devuelve una cadena vacía.
- Cuando el argumento pasado al método es null o undefined retornar un string vacío.
- Cuando la cadena tiene exactamente 1 caracter, devolver el mismo caracter.
- Cuando la cadena tiene más de 1 caracter, devolver la cadena invertida.
Exercise 2
Crear una clase ValidateArguments con un método Validate(args) que retorne un int.
Retorna: 1 si el usuario necesita información, 0 si está todo ok, -1 si la data/argumentos son invalidos.
Los datos pueden ser ingresados en minúscula o mayúscula.
- count: un entero entre 10 y 100 inclusive;
- name: una cadena de texto entre 3 y 10 caracteres inclusive;
- help: el usuario necesita información.
La forma que se pide los argumentos es la siguiente:
--name,Baufest,--count,25
Exercise 3
Dada una secuencia de N flechas para los 4 puntos cardinales: up ('^'), down ('v'), left ('<'), right ('>').
Desarrollar una funcion Solucion que enviandole un string (S) retorne el número mínimo de flechas que deben girarse para que todas apunten en la misma dirección.
Ej: Dado S = <<> retorna 1;
Dado S = <^^^ retorna 4
Dado S = <<< retorna 0
N debe ser un entero entre 1 y 100.
